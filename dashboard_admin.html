ระบบปฏิบัติงาน รปภ. — Admin
Role: ADMIN (ผู้ดูแลระบบ)

แดชบอร์ดผู้ดูแลระบบ

สำหรับผู้ดูแลระบบในการจัดการผู้ใช้, ไซต์งาน, โซน และดูภาพรวมข้อมูลการปฏิบัติงานของบริษัท (ในเฟสถัดไป)

ยินดีต้อนรับคุณ -
จัดการผู้ใช้ (Users)

ในเฟสถัดไปจะมีหน้าจอสำหรับจัดการบัญชีผู้ใช้ทั้งหมดของระบบ

  • เพิ่ม / แก้ไข / ปิดการใช้งานผู้ใช้
  • กำหนดสิทธิ์ User / SuperUser / Admin
  • ผูกพนักงานกับไซต์และโซนที่รับผิดชอบ
จัดการไซต์งาน / โซน

ใช้กำหนดข้อมูลไซต์ลูกค้า, จุดปฏิบัติงาน, และโซนที่ใช้แบ่งการดูแล

  • เพิ่ม/แก้ไขไซต์ลูกค้า
  • กำหนดโซนและความสัมพันธ์กับไซต์
  • ใช้เป็นฐานข้อมูลในการดึงรายงานโซน
รายงานภาพรวม

ดึงข้อมูลการลงเวลาเข้า–ออกของทั้งระบบ เพื่อใช้ตรวจสอบและวิเคราะห์ภาพรวม

  • รายงานเข้า–ออกทั้งบริษัท
  • Export CSV/Excel (เฟสถัดไป)
  • ใช้ตรวจสอบความเรียบร้อยในการปฏิบัติงาน
